To generate the SQL for the tables within flies, ensure that the maven 
build is up to date, then execute:

  mvn -Pjboss5,mysql,nogwt hibernate3:hbm2ddl

This must be run from within the 'flies-war' project. 

If running from the parent project, use this instead:
  mvn -Pjboss5,mysql,nogwt hibernate3:hbm2ddl -pl flies-war

The resulting schema is then available in:

  ./target/hibernate3/sql/schema.ddl

This should then be copied over src/etc/mysql/create_schema.sql
  cp ./target/hibernate3/sql/schema.ddl ./src/etc/mysql/create_schema.sql



Alternatively, if running from the root project:
  #mvn -Denv=dev clean install -DskipTests
